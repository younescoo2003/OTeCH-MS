## 📊 Project Overview

This Django web application provides a complete platform for clinicians, researchers, and patients to manage gait analysis data for Multiple Sclerosis patients. The system includes patient management, gait testing, news updates, marketplace for assistive devices, and user authentication.

### 🎯 Key Features by App

#### 🏥 **Patients App**
- Patient registration and management
- Medical history tracking
- Disease progression monitoring
- Patient profile management

#### 🧪 **Test_Gate App** (Core Feature)
- Gait analysis test management
- Test data collection and processing
- Gait parameter extraction
- Automated test reporting
- Test result visualization

#### 📰 **News App**
- Clinical research updates
- MS-related news and articles
- Announcement management
- Newsletter subscription

#### 🛒 **Marketplace App**
- Assistive device listings
- Medical equipment marketplace
- Product categories and filtering
- Vendor management

#### 👤 **Users App**
- Role-based access control
- Authentication system
- Doctor, Patient, Researcher profiles
- Account management
